# Formatiq-Konfigurator

Lokaler Next.js-Konfigurator fuer Stretch-Tent-Varianten.

## Wichtige Befehle

- `npm run dev` startet den lokalen Dev-Server
- `npm run check` fuehrt `lint` und `build` als Schnelltest aus
- `npm run build` erstellt den Produktions-Build

## Git-Workflow

- `git status` zeigt lokale Aenderungen
- `git add .` bereitet Aenderungen fuer den Commit vor
- `git commit -m "Nachricht"` speichert einen Stand lokal
- `git push` laedt den Stand zu GitHub hoch
